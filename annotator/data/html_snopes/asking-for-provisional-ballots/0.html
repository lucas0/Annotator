Provisional Voting Tutorial

Date

Tue, 05/19/2015 - 19:00

Provisional Voting - Election Administration Tutorial from Wis. Govt. Accountability Board on Vimeo.

Downloading Instructions

1. Click the link above, which will take you to the Vimeo website.

2. Click the 'Download' button that appears below the video.

 

3. Choose the "SD.MP4" file option and right-click on that link.

 

4. Another little menu will pop up. If you're a Windows user, choose 'Save as' or 'Save target as'.  Save the file to your desktop or somewhere else on your computer where you store saved files.  (If you're a Mac user, hold down the Control key and click the link, then choose 'Save link as' or 'Download linked file'.)
